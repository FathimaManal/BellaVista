# Bella Vista 
*A static Italian-themed restaurant website*  

## Overview  
Bella Vista is a modern and responsive **static website** for an Italian restaurant.  
Built using **HTML, Tailwind CSS, and JavaScript**, it provides a clean and aesthetic design to showcase the restaurant’s atmosphere, menu, and contact info.  

## Features  
-  **Responsive design** powered by Tailwind  
-  Italian-inspired color palette and layout  
-  Interactive navigation and menu sections with JavaScript  
-  Pages: Home, Menu, About, Reservations, Contact  

## Tech Stack  
- **HTML** – structure  
- **Tailwind CSS** – styling and responsiveness  
- **JavaScript** – interactivity
